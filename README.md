# git-intro-assignment-
Step-by-Step GitHub Assignment Instructions
Open Git Bash and navigate to the folder:
cd /c/Users/YourName/Documents/git-intro-assignment
Initialize local git repository:
git init
Stage all files:
git add .
Commit the changes:
git commit -m "Initial commit with README and sample file"
Link to GitHub repository (replace YOUR_USERNAME):
git remote add origin https ://github.com/YOUR_USERNAME/git-intro-assignment.git
Rename default branch to main and push to GitHub:
git branch -M main
git push -u origin main
Verify your files appear on GitHub.
Submit your GitHub repository link for the assignment.
