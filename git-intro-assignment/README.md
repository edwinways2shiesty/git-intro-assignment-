# Step-by-Step GitHub Assignment Instructions
 
1. Open Git Bash and navigate to the folder:
   ```bash
   cd /c/Users/YourName/Documents/git-intro-assignment
   ```
2. Initialize local git repository:
   ```bash
   git init
   ```
3. Stage all files:
   ```bash
   git add .
   ```
4. Commit the changes:
   ```bash
   git commit -m "Initial commit with README and sample file"
   ```
5. Link to GitHub repository (replace YOUR_USERNAME):
   ```bash
   git remote add origin https ://github.com/YOUR_USERNAME/git-intro-assignment.git
   ```
6. Rename default branch to main and push to GitHub:
   ```bash
   git branch -M main
   git push -u origin main
   ```
7. Verify your files appear on GitHub.
8. Submit your GitHub repository link for the assignment.
